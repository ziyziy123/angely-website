🔒 隐私政策

最后更新：2026年6月7日 · 生效日期：2026年6月7日

Angely（以下简称"本软件"）尊重并保护所有用户的个人隐私。本隐私政策说明了我们如何收集、使用、存储和保护您的信息。请仔细阅读。

一、信息收集

1.1 我们不收集的信息

  • 个人身份信息：我们不强制收集您的姓名、手机号、身份证号、地理位置、通讯录等任何可识别个人身份的信息。
  • 聊天内容：您的所有对话内容存储在本地设备上，未经您明确授权，绝不上传至云端服务器。
  • 设备敏感信息：我们不读取您的相册、文件系统、其他应用数据。

1.2 我们可能收集的信息

  • 匿名使用统计：应用启动次数、功能使用频率、崩溃日志（不包含任何个人可识别信息）。
  • API 调用数据：当您使用 AI 对话功能时，您输入的消息会通过您的 API Key 发送至 DashScope（阿里云）或您配置的 AI 服务商。此过程由您直接与 AI 服务商交互，我们不做中转存储。
  • Networking 数据：当您使用联网搜索功能时，搜索关键词会发送至搜索引擎。

二、信息使用

  • 匿名统计仅用于改善产品体验和修复 Bug。
  • 我们不会将任何数据出售、出租或分享给第三方。
  • 我们不会使用您的数据投放定向广告。

三、数据存储与安全

  • 本地存储：聊天记录、角色设定、设置偏好均存储在您设备的 localStorage 中，采用 AES-256 加密。
  • API Key：您配置的 DashScope API Key 加密存储在本地，仅用于调用 AI 服务。
  • 数据传输：所有网络通信使用 HTTPS 加密。

四、您的权利

  • 导出数据：您可以在设置中随时导出所有本地数据（JSON 格式）。
  • 删除数据：您可以在设置中一键清空所有本地数据。
  • 卸载即清除：卸载本软件后，所有本地存储数据一并删除。

五、第三方服务

本软件可能使用以下第三方服务：

  • 阿里云 DashScope：提供 AI 对话能力。其隐私政策见 阿里云隐私政策。
  • 搜索引擎：当您使用联网搜索功能时调用。

六、未成年人保护

本软件不针对 14 岁以下儿童设计。如果您是监护人且发现未成年人未经同意使用本软件，请联系我们。

七、隐私政策更新

我们可能会更新本隐私政策。更新后，我们会在软件内通知您。继续使用即表示同意更新后的政策。
