💻 安装指南

Windows

  • 下载 .exe 安装包，双击运行安装向导，按提示完成安装。
  • 或下载 .zip 绿色免安装版，解压后直接运行 Angely.exe。
  • 如遇 SmartScreen 提示，点击「更多信息」→「仍要运行」即可。
  • 安装包附带 MD5/SHA-256 校验码，建议安装前核验文件完整性。

macOS

  • 下载 .dmg 镜像文件，双击打开后将 Angely 拖入「应用程序」文件夹。
  • 首次打开如提示「无法验证开发者」：前往「系统设置 → 隐私与安全性」→ 找到 Angely → 点击「仍要打开」。
  • 此提示为 macOS Gatekeeper 标准安全机制，非软件安全问题。

Android

  • 下载 .apk 安装包，点击打开。
  • 系统可能提示「禁止安装未知来源应用」→ 点击「设置」→ 开启「允许来自此来源的应用」→ 返回安装。
  • 这是 Android 对非应用商店安装包的标准安全提示，安装完成后可重新关闭此权限。

🎙️ 语音设置

  • 开启语音：点击顶部状态栏的语音开关图标即可开启/关闭语音功能。
  • 语音对话：点击底部交互区的麦克风按钮，开始实时语音对话。
  • 全文朗读：长按任意 AI 回复消息，选择「朗读」即可播放。
  • 语速调节：设置 → 语音 → 拖动语速滑块（0.5x ~ 2.0x）。
  • 音调调节：设置 → 语音 → 拖动音调滑块。
  • 音量调节：设置 → 语音 → 拖动音量滑块，或使用系统音量键。
  • 情绪声线：AI 根据对话内容自动切换情绪声线（温柔/活泼/沉稳/幽默），无需手动设置。您也可以在角色设定中指定偏好的声线风格。

🎭 角色设定

  • 创建角色：设置 → 角色管理 → 新建角色 → 填写角色名称、身份、性格描述、语气风格。
  • 行为逻辑：可在高级设置中进一步定义角色的行为规则、知识范围、禁忌话题。
  • 切换角色：在聊天界面顶部点击当前角色名称，从列表中选择其他角色。
  • 修改角色：设置 → 角色管理 → 选择角色 → 编辑 → 保存。
  • 重置角色：设置 → 角色管理 → 选择角色 → 重置为默认。
  • 记忆说明：角色设定具备长期记忆，跨对话持续生效。如需清除特定记忆，请在聊天中说明或手动重置角色。

🌐 联网搜索

  • 开启搜索：点击底部交互区的搜索按钮，或输入框内输入 /search 关键词。
  • 自动总结：AI 会自动从搜索结果中提取关键信息，以自然口语化方式回复。
  • 断网提示：当网络不可用时，搜索按钮自动变灰并显示「网络不可用」提示。
  • 隐私说明：搜索请求经服务器转发，不记录搜索历史。详见《隐私政策》。

📋 常见问题

Q: 聊天记录保存在哪里？

聊天记录仅存储在您的本地设备上，默认使用 AES-256 加密。可在「设置 → 数据管理」中导出、备份或清空。

Q: 需要注册账号吗？

不需要。Angely 为完全本地化的应用，无需注册、无需登录即可使用全部功能。

Q: 有广告吗？

没有。Angely 承诺永久无广告、无弹窗、无捆绑推广。

Q: 支持哪些系统？

Windows 10+、macOS 12+、Android 8.0+。暂不支持 iOS 和 Linux。

Q: 如何反馈问题？

请发送邮件至 support@angely.app，附上问题描述、软件版本和系统信息。
