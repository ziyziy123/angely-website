📖 帮助中心

快速开始

  1. 下载安装：从官网对应平台下载安装包。
  2. 配置 API Key：首次启动时输入 DashScope API Key（获取地址，免费注册即有额度）。
  3. 开始聊天：输入消息，享受 AI 对话。

功能指南

🎙 语音对话

设置中开启"自动朗读"，AI 回复会自动语音播放。支持 5 种情绪音色：温柔、活泼、沉稳、幽默、自动。

😊 表情互动

点击输入框左侧 😊 按钮选择表情。AI 会根据对话内容自动匹配表情。

🌐 联网搜索

设置中开启搜索开关，AI 会自动联网获取最新信息。断网时自动禁用。

🎭 角色扮演

点击顶部 🎭 按钮创建自定义角色。设定身份、性格、语气后，AI 会按角色设定对话。

🎨 主题切换

点击 🌙/☀️ 按钮切换深色/浅色模式。设置中可选 5 种主题色。

常见问题

Q: 需要付费吗？

A: 软件完全免费。AI 对话通过您自己的 API Key 调用 DashScope，新用户有免费额度。

Q: 聊天记录存在哪里？

A: 所有数据仅在本地加密存储，不上传任何服务器。可在设置中导出或清空。

Q: 支持哪些平台？

A: Windows（.exe）、macOS（.dmg）、Android（.apk）。

Q: macOS 安装提示"无法验证开发者"怎么办？

A: 前往「系统设置 → 隐私与安全性」→ 点击「仍要打开」即可。

Q: Android 安装提示"未知来源"怎么办？

A: 点击「设置 → 允许安装未知来源应用」→ 返回安装即可。这是 Android 标准安全提示。
